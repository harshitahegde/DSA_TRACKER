#  DSA_TRACKER

**DSA_TRACKER** is a professional Flutter-based application designed to help developers track and manage their Data Structures and Algorithms (DSA) practice. This tool provides a centralized dashboard to monitor daily progress, time coding sessions, and visualize problem-solving statistics.

## ‚ú® Key Features
* **Progress Monitoring:** Keep a detailed log of completed problems to ensure consistency.
* **Integrated Timer:** Use the built-in timer for focused, "deep work" coding sessions.
* **State Management:** Built using the `Provider` package for robust and efficient data handling across the app.
* **Data Visualization:** A dedicated statistics screen to help you analyze your learning patterns and success rates.

## üìÇ Project Structure
* **lib/**: The core of the application containing all Dart logic and UI.
  * `main.dart`: The entry point of the application.
  * `question_provider.dart`: Manages the state and logic for DSA questions.
  * `stats_screen.dart`: The UI responsible for displaying progress statistics.
* **pubspec.yaml**: Configuration file listing all project dependencies and assets.
* **android/ / ios/ / web/**: Platform-specific configurations to allow cross-platform deployment.
